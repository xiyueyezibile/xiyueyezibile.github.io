# 10-DOCTYPE 和 lang 以及字符集的作用

文本类型声明，作用是告诉浏览器使用哪种HTML版本来显示网页，必须写到最前面 意思是当前页面采用HTML5版本来显示网页

lang 语言种类 用来定义当前文档显示的语言：
en 为英语
zh-CN 为中文
fr 为法语
主要是告诉浏览器是什么网站，定义为英文也可以写中文

兄弟关系，用➕加 tab
用。加类名，自动生成 div class=“类名”
