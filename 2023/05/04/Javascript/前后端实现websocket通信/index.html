# wss.clients

wss.clients 方法中存储所有的连入对象，保存为一个数组，通过它可以遍历所有连入的客户端发送消息

# wss.on(‘connection’,callback(ws))

有客户端连接时会调用，ws 为连接进来的客户端对象

# ws.on(‘message’,callback(message))

接受到客户端 / 服务端消息时调用，message 为信息

# ws.send(message,callback(err))
