REST

REST表示层状态的传输

请求的方法：

GET 加载数据

POST 新建或添加数据

PUT 添加或修改数据

PATCH 修改数据

DELETE 删除数据

OPTION 由浏览器自动发送，检查请求的一些权限

API(接口) Endpoint(端点)

postman是一个软件，可以向服务器发送各种请求，帮助我们测试API

AJAX

在js中，发送请求加载数据的技术叫AJAX

​ AJAX

​ -A 异步 J JavaScript A and X XML

​ -作用：通过js向服务器发送请求加载数据

​ -可选择发送请求的方案:

​ 1 XMLHTTPRequest(xhr)基本淘汰

​ 2 Fetch

​ 3 Axios

xhr

fetch

fetch是xhr的升级版，采用promise API

作用和AJAX一样，但使用起来更加友好

fetch是原生js就支持的一种方式

fetch(“请求路径”,配置对象{

​ method:”get”, //发送方式

headers:{//请求头

“Content-type”:”application/json”

},

//通过body发送数据时，必须通过请求头确定数据类型

body:JSON.stringify({}) //请求体，给服务器添加什么数据

})

2 引用

baseURL 自动服务器根目录

url 请求地址

data:{} 请求体

params:{}指定查询字符串

timeout: 过期时间单位毫秒

transformRequest:[function(data,headers) {

headers[“Content-Type”] = “”

CORS(跨域资源共享)

跨域请求：

​ 如果两个网站的完整域名不相同

​ a网站：http://haha.com

​ b网站：http://heihei.com

​ 跨域需要检查3个东西：

​ 协议 域名 端口号

​ 只要有一个不同，就算跨域

​ 当通过AJAX发送跨域请求时，浏览器为了服务器的安全

​ 会阻止JS读取到服务器的数据

​ 解决方案：

​ 服务器中设置一个允许跨域的头

​ Access-Control-Allow-Origin

​ 允许哪些客户端访问我们的服务器

本地存储

sessionStorage 中存储的数据页面一关，数据就会丢失

localStorage 存储时间比较长

setItem()用来存储数据

getItem()取数据

removeItem() 删除数据

clear() 清空
